# User Guide Screenshots

This folder contains all screenshots for the user guide page.

## Required Screenshots

Add the following screenshot files to this directory:

### 1. Setup & Installation
- **File:** `setup-installation.png`
- **What to capture:** Screenshot of the plugin installation process or the plugin in the Front sidebar

### 2. Conversation Locations
- **File:** `conversation-locations.png`
- **What to capture:** Map showing locations detected from a Front conversation with markers

### 3. Conversation Filter
- **File:** `conversation-filter.png`
- **What to capture:** The conversation filter dropdown showing options (Current, All Conversations, etc.)

### 4. Location Search
- **File:** `location-search.png`
- **What to capture:** The search bar with search results dropdown

### 5. Create Pinpoints
- **File:** `create-pinpoints.png`
- **What to capture:** Map with custom pinpoints and the naming dialog/popup

### 6. Create Routes
- **File:** `create-routes.png`
- **What to capture:** Map showing a route with the directions panel displaying turn-by-turn instructions

### 7. Transportation Modes
- **File:** `transportation-modes.png`
- **What to capture:** The transportation mode selector showing all options (car, walk, bike, plane, boat, container ship)

### 8. Speed & Cost Settings
- **File:** `speed-cost-settings.png`
- **What to capture:** The speed settings panel showing editable speeds and costs for different transport methods

### 9. Cost Calculation
- **File:** `cost-calculation.png`
- **What to capture:** Directions panel showing a route with cost breakdown displayed

### 10. Name Points
- **File:** `name-points.png`
- **What to capture:** Dialog/popup showing how to name a point or pinpoint

### 11. Map Styles
- **File:** `map-styles.png`
- **What to capture:** The map layer switcher showing Road, Terrain, and Satellite options (or a composite showing all three views)

## Image Guidelines

- **Format:** PNG or JPEG
- **Resolution:** At least 1200px wide for clarity
- **Aspect Ratio:** 16:9 or 4:3 works best
- **File Size:** Keep under 500KB per image for fast loading
- **Quality:** Use high quality but compress for web

## Optional

You can also add:
- A favicon: `favicon.png` or `favicon.ico`
- Logo: `logo.png`
- Hero image: `hero.png`
